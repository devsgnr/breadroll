---

#### `Breadroll.open.json<T>(...)`

This function casts imported JSON object into the required type; from an API call or on disk and returns a dataframe, **Note**: the JSON needs to have a shape of `Array<Record<string, unknown>>{:ts}`

Parameter

- `object: Record<string, never> | {}` `as any` - the location of the file.

Returns

- `Dataframe<T>`

```typescript
import data from "/path/to/file.json";
const csv: Breadroll = new Breadroll({ header: true, delimiter: "," });

const df: Dataframe<T> = csv.open.json<T>(data);
```
