#### `Dataframe.value`

This function exposes (ie. returns) an array of objects `Array<Record<string, unknown>>` that is within the `Dataframe`

Returns

- `Array<Record<string, unknown>>`

```typescript
const values: Array<Record<string, unknown>> = df.value; // [{}, {}, {}, ...]
```
