#### `Dataframe.dtypes`

This function return an object `Record<string, string>{:ts}` that shows the data type of each column in the dataframe in a key:value pair

Returns

- `Record<string, string>{:ts}`

```typescript
const dtypes: Record<string, string> = df.dtypes; // { age: "number", ... }
```
